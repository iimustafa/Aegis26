# ⚽ Aegis26 — FIFA World Cup 2026 AI Travel Planner

A modern AI-powered FIFA World Cup 2026 travel planning platform using FastAPI, LangGraph, Machine Learning, and an LLM reasoning layer.

## 🚀 Features

- AI Travel Planning
- Multi-Agent Architecture
- Real ML Match Prediction
- Analytics Dashboard
- Interactive UI/UX
- Real FIFA 2026 Fixtures Validation

## 🧠 Tech Stack

### Backend
- FastAPI
- LangGraph
- Scikit-learn
- Pandas
- Joblib

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS
- Recharts

### AI / LLM
- Ollama
- LangChain
- Tavily Search API

# 🧩 System Pipeline

```txt
User Request

    ↓

Analytics Agent
```

# ⚙️ Installation

## Backend

```bash
python -m venv langgraph_env2
source langgraph_env2/bin/activate
pip install -r requirements.txt
```

## Frontend

```bash
npm install
npm run dev
```
